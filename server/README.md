## backend (express + prisma)

0. llenar .env

```bash
code .env
# completar con credenciales
```

1. instalar dependencias

```bash
npm i
```

2. generar bloat de express

```bash
npm run express:generate
```

3. ejecutar migracion de prisma

```bash
npm run db:migrate
```

3. correr

```bash
npm run dev
```
